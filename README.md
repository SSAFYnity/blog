# SSAFYnity 블로그

> 싸피 수료생들이 직접 운영하는 동문회의 공식 블로그

**[ssafynity.github.io/blog](https://ssafynity.github.io/blog)**

기술, 활동 후기, 공지사항을 포함한 SSAFYnity의 공식 블로그입니다.

## 스택

- Astro
- GitHub Pages + GitHub Actions

## 카테고리

| 카테고리 | 설명 |
|----------|------|
| 기술 | 기술 관련 포스팅 |
| 활동 | 행사 후기, 회고 |
| 후기 | 회원 후기 |
| 공지 | 공지사항 아카이브 |
| 회비 | 연도별 회비내역 |

## 포스팅 작성

1. 이슈를 생성합니다 (`카테고리:*` 라벨 부착).
2. `src/content/카테고리/YYYY-MM-DD-slug.md` 파일을 작성합니다.
3. Pull Request를 열고 리뷰를 요청합니다.
4. `main` 브랜치에 머지되면 자동 배포됩니다.

## 시작하기

```bash
npm install
npm run dev
```
